#RECONBOT

DESCRIPTION : 

ReconBot is a tactical simulation app that demonstrates robotics and AI concepts in a visually engaging way. It features:

Secure login system

Terrain scanning and camouflage adaptation

Friend‑or‑foe detection

Sequential firing and reloading with sound effects

Kill confirmation log and HUD interface

How to Run the App:
